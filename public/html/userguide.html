Patrimoine


Les objets


Dans patrimoine, il est possible d'ajouter nos possessions.
Nous pouvons naviguer dans nos possessions grâce à une liste d'objet.
Le nom d'objet en noir est l'objet sélectionné. L'objet au dessus,
s'il y a, est l'objet contenant l'objet que vous regarder, tandis que
les objets au dessous sont les objets qu'il contient.

Ajouter



Le bouton d'ajout d'objet nous permet de faire apparaître une page
nous demandant de remplir les informations du nouvel objet à ajouter.

Supprimer



Le bouton de suppression d'un objet permet de supprimer un objet.

Modifier



Le bouton de modification nous permet de faire apparaître une page
nous demandant d'entrer les informations de l'objet sélectionné à modifier.

Déplacer



Le bouton de déplacement nous permet de faire apparaître une page
nous demandant dans quel objet nous voudrions le déplacer.

Émettre un rapport



Le bouton de rapport nous permet de produire un rapport montrant
la valeur totale d'un objet et de ses composants.

Prêter



Le bouton de prêt nous permet de prêter un objet à un contact.

Code QR



Le bouton de Code QR génère un code qui permet de lire des informations sur l'objet

Ajouter une image à un objet



Le bouton d'ajout d'image nous ouvre une fenêtre nous permettant d'ajouter une image
à un objet.

Ajouter une zone réactive à une image d'un objet



Le bouton d'ajout d'une zone réactive permet d'ajouter des zones dans les images
faisant un lien vers un autre objet.

Supprimer une image d'un objet



Le bouton de suppression d'une image permet de supprimer une image d'un objet.
